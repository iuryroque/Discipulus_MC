# Documentação da API

O projeto já possui integração com OpenAPI (Swagger UI e ReDoc) via springdoc-openapi.

- **Swagger UI:** Acesse em: `http://localhost:8080/swagger-ui.html`
- **ReDoc:** Acesse em: `http://localhost:8080/redoc`

Essas ferramentas permitem visualizar, testar e mapear todas as APIs REST do sistema automaticamente.
